# 🛡️ RPG Admin Balance System (Система Адміністрування Ігрового Балансу)

![Java Version](https://img.shields.io/badge/Java-21-orange?style=for-the-badge&logo=openjdk)
![JavaFX Version](https://img.shields.io/badge/JavaFX-21-blue?style=for-the-badge&logo=javafx)
![Build Tool](https://img.shields.io/badge/Maven-3.9+-red?style=for-the-badge&logo=apachemaven)
![Database](https://img.shields.io/badge/H2-Embedded-darkblue?style=for-the-badge)

**RPG Admin Balance System** — це десктопний додаток для гейм-дизайнерів та адміністраторів RPG-ігор, який дозволяє гнучко керувати ігровим світом, балансувати економіку гравців, переглядати журналювання дій та візуалізувати ігрову карту у реальному часі. 

Додаток розроблено з використанням багатошарової архітектури на мові **Java 21** та фреймворку **JavaFX**.

---

## 🚀 Ключовий функціонал (Features)

*   **🔐 Аутентифікація та Безпека**:
    *   Безпечний вхід та реєстрація адміністраторів.
    *   Хешування паролів з використанням сучасного алгоритму **BCrypt**.
    *   Симуляція надсилання верифікаційного листа на Email при реєстрації.
*   **👥 Управління гравцями**:
    *   Перегляд повного списку гравців у формі інтерактивної таблиці.
    *   Розширений динамічний пошук та фільтрація гравців за ігровими параметрами (золото, досвід, рівень, ім'я).
    *   Можливість ручного коригування балансу валюти та енергії.
*   **🗺️ Візуалізація та імпорт карти**:
    *   Парсинг ігрової карти у форматі JSON за допомогою власного парсера `SimpleJsonParser`.
    *   Ефективний **пакетний імпорт (Batch Insert)** великих обсягів даних в БД для оптимізації продуктивності.
    *   Динамічна візуалізація сітки карти 100х100 клітинок за допомогою **JavaFX Canvas API**.
*   **💎 Балансування ігрового процесу**:
    *   Управління скарбами (Treasures) та ігровими подіями (Events).
*   **📜 Аудит та Логування**:
    *   Перегляд журналів дій (Action Logs) для відстеження операцій, здійснених через панель адміністрування.

## 🛠️ Технологічний стек (Tech Stack)

*   **Мова**: Java 21
*   **Графічний інтерфейс**: JavaFX 21.0.1 (FXML)
*   **База даних**: Вбудована H2 Database (File-based)
*   **Міграції БД**: Flyway Core (керування схемами та версіонуванням бази даних)
*   **Доступ до даних**: Native JDBC API + **Власний Thread-Safe Connection Pool** на базі `BlockingQueue`
*   **Допоміжні інструменти**: Lombok (генерація геттерів, сеттерів, білдерів), Jakarta Mail API (емуляція відправки пошти)
*   **Збірка проєкту**: Apache Maven

## 📂 Архітектура проєкту

Проєкт побудований відповідно до принципів чистої архітектури та розділення обов'язків:
*   `com.minden.entity` — опис бізнес-сутностей (Domain Entities).
*   `com.minden.repository` — абстракції та реалізації взаємодії з базою даних (шаблон **Repository**).
*   `com.minden.service` — шар бізнес-логіки додатка (шаблон **Service**).
*   `com.minden.ui` — контролери JavaFX та описи екранів додатку.
*   `com.minden.infrastructure` — сервіси хешування паролів та роботи з мережею/поштою.
*   `com.minden.util` — утиліти парсингу JSON, валідації вхідних даних тощо.

## 📦 Інструкція зі встановлення та запуску

Ви можете обрати найбільш зручний спосіб запуску системи: використати вже готовий реліз (найпростіше) або зібрати додаток самостійно з вихідного коду.

---

### 🚀 Спосіб 1. Швидкий запуск готового релізу (Для користувачів)

У нашому проєкті налаштовано автоматичне пакування у нативні інсталятори через GitHub Actions. Вам не потрібно мати встановлену Java чи Maven для цього способу!

1. Перейдіть на вкладку **[Releases](https://github.com/Minden20/adminbalancesystem/releases)** у цьому репозиторії.
2. Завантажте необхідний файл для вашої операційної системи:
   *   🪟 **Windows (10/11)**: Завантажте інсталятор `.msi`. Запустіть його та слідуйте інструкціям. Додаток буде встановлено в систему, а на робочому столі та в меню "Пуск" з'явиться ярлик **RPG Admin Menu**.
   *   🐧 **Linux (Ubuntu/Debian)**: Завантажте пакет `.deb`. Встановіть його за допомогою терміналу:
       ```bash
       sudo dpkg -i rpg-admin-menu_*.deb
       ```
3. Просто запустіть додаток з вашого меню програм!

---

### 👨‍💻 Спосіб 2. Запуск з вихідного коду (Для розробників)

Якщо ви розробник і плануєте редагувати код проєкту, скористайтеся цією інструкцією.

#### Вимоги
1. **Java Development Kit (JDK) версії 21**.
2. **Apache Maven версії 3.8** або новішої.

#### 1. Клонування та компіляція
Відкрийте термінал і виконайте:
```bash
git clone https://github.com/YOUR-USERNAME/adminbalancesystem.git
cd adminbalancesystem
mvn clean compile
```

#### 2. Запуск у режимі розробки
Запустити графічний інтерфейс напряму:
```bash
mvn javafx:run
```

#### 3. Збірка автономного виконуваного .jar файлу
Щоб запакувати додаток у Fat JAR (всі залежності включено):
```bash
mvn clean package
```
Після виконання готовий файл з'явиться в каталозі `target/rpgadminmenu-1.0-SNAPSHOT.jar`.
Запустити його можна командою:
```bash
java -jar target/rpgadminmenu-1.0-SNAPSHOT.jar
```
